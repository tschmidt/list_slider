jQuery Plugin: list_slide

Description

Show a list that will display additional information for each item. View source to see the structure for each item in the list.

Options

Option Description
speed The speed in which the animation completes. This is represented in milliseconds
showText The text that should be displayed for the link to "show" the additional information
hideText The text that should be displayed for the link to "hide" the additional information

HTML Structure

For this plugin the HTML structure that is expected is the following: 

<ol id="people">
  <li>
    <div class="main">
      <div class="img">
        <img src="source/to/img" />
      </div>
      
      <div class="info">
        <span>Name: Jon Dough</span>
        <span>Other: Long walks on the beach</span>
      </div>
    </div>
    
    <div class="additional">
      <ol>
        <li>One</li>
        <li>Two</li>
        <li>Three</li>
      </ol>
    </div>
  </li>
</ol>

Examples

  1. Name: Jon Dough Other: Long walks on the beach
    1. One
    2. Two
    3. Three
  2. Name: Jane Dough Other: Eating Bon Bons with Mr. Winkles
    1. One
    2. Two
